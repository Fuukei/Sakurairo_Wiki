# 部署WordPress

将WordPress的文件上传到服务器中

将上传的压缩包解压到网站根目录

访问需要安装的网站域名

输入我们的数据库进行安装必备库

输入我们的一些站点信息和管理员账号

干得不错！来登录后台吧！

# 上传并启用主题

进入仪表盘，点击左侧菜单的 外观 选项

点击主题页面的 安装主题

在刷新的页面点击 上传主题

然后的 上传主题 中，选中 桜いろ 主题安装

安装之后，启动主题
